ALL 8 SWITCHES WIRED THE SAME WAY:
-----------------------------------

Each SPDT switch has 3 terminals:
  - Terminal 1: Connected to Signal Wire (Green/Yellow/Brown/etc.)
  - Terminal 2: Connected to +5V (Red wire) - This is the ON position (0)
  - Terminal 3: Connected to GND (White wire) - This is the OFF position (1)

When switch is ON (0):  Signal wire → +5V
When switch is OFF (1): Signal wire → GND

Common Connections:
  - Red wire (+5V): Connect to Terminal 2 of ALL switches
  - White wire (GND): Connect to Terminal 3 of ALL switches


IMPORTANT NOTES:
----------------

1. MUST USE SPDT SWITCHES - Do NOT use simple ON/OFF switches!
   The cartridge's multiplexer chip requires signals to be either
   +5V or GND, never floating (unconnected).

2. Switch Logic:
   - ON position = 0 (binary) = Connect signal to +5V
   - OFF position = 1 (binary) = Connect signal to GND

3. Example: To select game #5 (KOF2002) with binary code 00000101:
   Switch 1: ON  (0) → +5V
   Switch 2: OFF (1) → GND
   Switch 3: OFF (1) → GND
   Switch 4: OFF (1) → GND
   Switch 5: OFF (1) → GND
   Switch 6: ON  (0) → +5V
   Switch 7: OFF (1) → GND
   Switch 8: ON  (0) → +5V

4. The switches are read from right to left (LSB to MSB):
   Switch 1 = LSB (Least Significant Bit)
   Switch 8 = MSB (Most Significant Bit)
